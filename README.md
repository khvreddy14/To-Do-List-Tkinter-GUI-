# To-Do List App

A simple desktop To-Do List application built using Python and Tkinter.

## Features

- Add tasks
- Delete selected tasks
- Clear all tasks
- Automatic task saving
- Load saved tasks on startup
- User-friendly GUI

## Technologies Used

- Python
- Tkinter

## Installation

No external libraries required.

## Run

```bash
python main.py
```
